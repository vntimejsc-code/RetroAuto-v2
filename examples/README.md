# Example Projects

This directory contains example RetroAuto scripts.

## Available Examples

### notepad_demo/
A simple script that demonstrates:
- Opening Notepad via Win+R
- Typing text (with Vietnamese support)
- Saving a file
- Closing the application

**To run:**
1. Open RetroAuto
2. File → Open → Select `examples/notepad_demo/script.yaml`
3. Press F5 or click Run

## Creating Your Own Projects

1. Create a new folder for your project
2. Create `script.yaml` with your automation flow
3. Create `assets/` folder for template images
4. Use the Capture tool to create image templates
