## üåç Global Hotkeys (Work Everywhere)

| Hotkey | Function | Description |
|--------|----------|-------------|
| `F4` | Capture Coordinates | Save current mouse position to Coordinates Panel |
| `Ctrl+Space` | Capture Coordinates | Alternative for F4 |
| `Ctrl+Shift+C` | Screen Capture | Open capture overlay for asset capture |
| `Ctrl+Shift+S` | Stop Script | Stop running script immediately |

> **Note:** Global hotkeys work even when the app is minimized or another window has focus.

## üí° Tips

1. **Focus matters for local shortcuts** - Click on the panel first before using panel-specific shortcuts
2. **Global hotkeys always work** - F4, Ctrl+Space, Ctrl+Shift+C/S work from anywhere
3. **Multi-select** - Use `Ctrl+A` then `Del` to quickly clear all items
4. **Quick workflow** - Use `F4` to capture coordinates while positioning mouse, then add them to script
